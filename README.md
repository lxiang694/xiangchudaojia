# 乡厨到家 · 上门做饭服务网站

一个面向“家乡味、上门做饭、健康餐、家宴酒席、城市合伙人”的响应式官网原型。

## 本次 UI 优化重点

- **视觉升级**：采用暖橙色 + 米白色 + 家庭厨房感的设计语言，增强“家乡味、可信赖、温暖”的品牌感。
- **首页结构重构**：从用户痛点 → 服务场景 → 智能预约 → 推荐厨师 → 套餐 → 流程 → 合伙人，形成更清晰的转化路径。
- **移动端优先**：适配手机浏览，底部固定“看厨师 / 立即预约”快捷入口。
- **预约卡片优化**：日期、人数、服务类型、食材方式集中在同一模块，并自动生成预估价格。
- **厨师卡片优化**：突出厨师头像、擅长菜系、标签、服务场景，增强用户选择信任感。
- **交互动效**：增加滚动入场、预约估价、移动端菜单、轻提示等基础交互。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

如果要部署到 GitHub Pages：

1. 打开仓库 Settings
2. 找到 Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main` / root
5. 保存后等待 GitHub 生成访问链接
