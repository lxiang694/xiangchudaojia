# 双宝成长学习系统

这是一个面向两个孩子的成长型学习系统网站原型，覆盖 **幼儿园大班到六年级**，重点不是单纯刷题，而是通过每日规划、语音陪学、英语跟读、星星奖励、AI总结和科普实践，培养孩子的学习兴趣、自信心和主动学习能力。

## 我对当前项目的理解

这个项目适合做成一个 **儿童学习系统 / 家长陪学工具**：

- 小儿子：当前幼儿园大班，重点是兴趣启蒙、慢速语音、看图表达、数字感和英语听读。
- 大儿子：当前二年级下学期，今年秋季进入三年级，重点是二升三衔接、阅读理解、万以内计算、英语句型、科普探索和自我规划。
- 长期路线：从大班、一年级、二年级、三年级、四年级、五年级到六年级，逐步提高难度。

## 本次 UI 设计方向

- **学习系统定位**：整体改为“学习乐园 + 成长规划系统”，不再是普通官网。
- **首页主视觉**：突出“今天学什么、下一步做什么”，让孩子能理解进度。
- **双孩子入口**：首页可切换“弟弟 · 大班”和“哥哥 · 二升三”。
- **每日15题机制**：语文5题、数学5题、英语5题，适合每天坚持。
- **全学科模块**：语文森林、数学岛、英语星球、AI与宇宙探索、运动与社会实践。
- **大班与英语语音增强**：支持中文慢速播报、英语读题、慢速跟读、完成后语音提醒休息。
- **AI总结卡片**：完成练习后展示答对、答错、复习提醒和下一进度。
- **年级路线图**：覆盖大班到六年级，每个阶段都有语文、数学、英语和综合能力目标。
- **移动端适配**：适合 iPad、手机和电脑浏览。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

如果在 iPad 上打开，语音通常需要先点击一次页面按钮，例如“播报计划”或“测试英语跟读”，浏览器才允许播放声音。
